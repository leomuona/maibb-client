# maibb-client

MaiBB client

- Node
- Typescript
- React

## Commands

First make your .env and make changes according to your local setup:

```
cp .env.example .env
```

For local development, use this:

```
npm start
```

Below are some other commands.

Check linting rules:

```
npm run lint
```

Format files with linting rules:

```
npm run format --write
```

Compile:

```
npm run build
```
